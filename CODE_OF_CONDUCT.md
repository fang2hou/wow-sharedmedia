# Code of Conduct

## Our Pledge

We want participation in this project to be respectful, constructive, and welcoming for everyone.

## Our Standards

Examples of behavior that contribute to a positive environment include:

- showing empathy and respect
- giving actionable technical feedback
- assuming good intent while communicating clearly
- focusing on the work rather than personal attacks

Examples of unacceptable behavior include:

- harassment or discriminatory language
- bad-faith review behavior
- personal attacks or insulting remarks
- publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that do not align with this Code of Conduct.

## Scope

This Code of Conduct applies within project spaces and in public spaces when someone is representing the project.

## Reporting

If you experience or witness unacceptable behavior, please contact the maintainers through the repository's private contact channels.
